# team-calendar-social

Create a team calendar application with social media authentication with monthly, weekly and daily views, to create, update and delete personal and team events, and use a backend REST API for all operations.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/team-calendar-social.git
cd team-calendar-social
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
